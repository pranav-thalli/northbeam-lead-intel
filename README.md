# Northbeam Lead Intel Dashboard

Automated competitive intelligence & lead enrichment pipeline that scrapes 25 DTC brand websites for ad pixels, attribution tools, and marketing tech, then scores each brand as a Northbeam prospect.

## What the Data Reveals

**48% of DTC brands have no attribution tool at all.** Of 23 brands successfully scraped, 11 have zero attribution tooling despite running multi-channel ad campaigns across Meta, Google, TikTok, and Pinterest. These are greenfield prospects -- no competitor displacement needed.

**Northbeam already leads the DTC attribution market.** 6 brands in our sample already use Northbeam, compared to 3 for Triple Whale and 1 for Rockerbox. The competitive moat is real, and the GTM focus should be on converting the 48% with no tool rather than displacing competitors.

**91% of prospects run on Shopify** (21 of 23 brands), where Northbeam has a native integration. This means low-friction onboarding for nearly every prospect in the pipeline.

## Architecture

```
brands.json (25 DTC brands)
      |
      v
scripts/scrape.py (Python pipeline)
  - Concurrent HTTP scraping
  - Regex-based tech detection (ad pixels, martech, attribution, platform)
  - Meta Ad Library checks
  - Multi-factor lead scoring (0-100)
      |
      v
public/data/leads.json
      |
      v
Next.js Dashboard
  - Insight cards (above the fold)
  - Filterable lead table (tier, attribution status, ad channel count)
  - Lead detail view (score breakdown, detected tech, Meta Ad Library link)
  - Market analysis tab (competitor chart, martech adoption, methodology)
```

## Tech Stack

- **Pipeline**: Python 3, stdlib only (no external dependencies)
- **Frontend**: Next.js 16, React 19, Tailwind CSS 4, TypeScript
- **Data**: Static JSON generated by pipeline, served as static asset
- **Deployment**: Vercel

## Getting Started

```bash
# Run the scraping pipeline
python3 scripts/scrape.py

# Install frontend dependencies
npm install

# Run the dashboard
npm run dev
```

The pipeline outputs to `public/data/leads.json`, which the Next.js app reads at build time.

## Data Source

All data is scraped live from real eCommerce websites. The pipeline detects technology by pattern-matching JavaScript includes, pixel tags, and platform signatures in the page HTML. No APIs or credentials required -- this runs on publicly available information only.

Brands analyzed: Allbirds, Glossier, Gymshark, Ridge, MVMT, Bombas, Brooklinen, Warby Parker, Away, Casper, Fabletics, Mejuri, Chubbies, Rothy's, Parade, Alo Yoga, Vuori, Skims, True Classic, HexClad, Jones Road Beauty, Obvi, Cozy Earth, Cuts Clothing, Native.
